# 🚀 GrowthPilot AI
### AI-Powered E-commerce Growth Assistant

GrowthPilot AI is an intelligent business analytics platform that helps e-commerce businesses make smarter decisions using Artificial Intelligence and Machine Learning.

Built for the **NetElixir AI Hackathon 2026**, GrowthPilot AI transforms raw sales and inventory data into actionable business insights through forecasting, customer analytics, marketing intelligence, and an AI-powered business copilot.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sahilgandhi413-alt/growthpilot-ai.git
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Machine Learning

Train Model

```bash
python src/train_model.py
```

Generate Features

```bash
python src/generate_features.py
```

Predict

```bash
python src/predict.py
```

# 🎯 Problem Statement

Small and medium-sized e-commerce businesses often struggle to make informed decisions because business data is scattered across multiple systems.

GrowthPilot AI solves this by combining AI, analytics, forecasting, and business intelligence into a single unified platform.
